Image Search API

Input a search query to see the top ten images from Google Image Search.

https://colin-fcc-image.herokuapp.com/api/Tokyo

The response will be an array of objects in the following format.

{
"title": "Views of Tokyo",
"url": "www.photos.com/cool-tokyo-photo.jpg",
"context": "www.photos.com"
}

Optionally, you can specify an offset to skip through results by appending ?offset=5 to the end of your search, where 5 can be any number.

You can view the ten most recent search queries by visiting https://colin-fcc-image.herokuapp.com/recent. The response will be an array of objects in the following format.

{
"query": "Shiba",
"time": "Fri Dec 08 2017 20:04:46 GMT+0900 (JST)"
}
